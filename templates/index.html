模版赋值

有时候我们需要对一个数组赋值,就和php中赋值是一样的
{($m[]="你好啊")}
数组输出：{($m.0)}

如果不是多维数组，可以用.号简写, $m.0等于$m[0],当然也可以用$m[0],多维数组就必须要用[]，比如$m[0][1]，不能写成$m.0.1



循环语法

循环函数是我们用的最为普遍的函数之一,在数据库分页和显示方面,它起着巨大的作用

foreach循环:

这个函数我们一般是用来遍历数组的 在模版中有两种使用方法

例子一

{($arr=array(1,2,3,4,5))} {(foreach $arr)} {($key)}:{($value)}
{(/foreach)}
以上只写了一个参数,foreach可以有三个参数的,这是种简单写法,在内部循环中,$key代表键值,$value代表数组的值,这是默认定义的,你可以自己定义,看例子二

例子二

{($arr=array(1,2,3,4,5))} {(foreach $arr as $a=>$b)} {($a)}:{($b)}
{(/foreach)}

以下其他循环的使用

while循环

例子:
{($i=0)} {(while $i<10)} {($i++)}, {(/while)}

for循环

例子:
{(for $i=0;$i<10;$i++)} {($i)}, {(/for)}

IF判断

if判断

标签中的else不要加/

{($f=123)} {(if isset($f))} $f不为空 {(/if)}

{(if isset($s))} $s变量存在 {(else)} $s变量不存在 {(/if)}

获取全局变量

{(if isset($get['a']) && $get['a']==null)} $get['a']等于null

点击此次改变$get['a']的值为123 {(elseif isset($get['a']) && $get['a']==123)} $get['a']等于123

点击此次改变$get['a']的值为空 {(else)} 模板中的$get['a']就代表$GET['a'],$post就代表$POST，这些都是全局变量

点击此次改变$get['a']的值为123 {(/if)}


  • $post => $_POST
  • $get => $_GET
  • $cookie => $_COOKIE
  • $session => $_SESSION
  • $files' => $_FILES
  • $server => $_SERVER
  • $this => $this
  • $cpu => self::cpu()
